#### ***函式***  
* ***函式目的***  
1. 函式為具有名稱的一群指令->為提供程式多次呼叫 (function call)，以避免重複撰寫程式碼  
2. 將相關的程式指令集合再一起，可讓程式更為模組化 (Modularization)  

* ***函式的語法***  
```
def <function_name>(parameters):  
    <body>   #記住本體要縮排
```
```
ex:  
def printsum(a,b):  
    print("兩數之和:",a+b) ＃可用,或+隔開數值與字串差別

a=30
b=50 
printsum(a+2,b+5)
```
